# Luca's Banking Project
This is the repository for my banking project in ELEC 1520
## Building
In a code-space or local machine, run `make` in your terminal to build the project into `Build.out`

***IF: You do not have make installed on your machine, you can run `g++ *.cpp -o Build.out` to achieve the exact same result***
## Running 
On any machine you compile on, you should be able to simply run `./Build.out` in the same directory you build in. If you have any issues building and running, please email me at luca.pendleton@ucdenver.edu
